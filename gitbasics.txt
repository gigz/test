Clone a repository: git clone git://github.com/...
Create a new repository from the current folder: git init
Start tracking new file: git add
Remove tracking of a file: git rm
Stage changes to the file: git add
Unstage changes: git reset
See the status of the changes: git status
See modified diff: git diff
See staged diff: git diff --staged
Commit: git commit -m "Changelist description"
Commit without staging: git commit -a
View change history: git log
View last 2 changes with diff: git log -p -2
Undo local modification: git checkout -- <filename>
